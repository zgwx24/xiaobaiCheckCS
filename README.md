## 🎮 如何进入 UB 社区服进行检视？

复制好指令后，你需要进入支持皮肤检视的社区服（如 UB 社区服）。以下是三种常见的进入方式：

### 1\. 通过 UB 官网进入

*   浏览器访问 [UB 社区服官方网站](http://csgo.moeub.cn/welcome)。
    
*   在服务器列表中挑选一个延迟低、有空位的服务器。
    
*   点击“一键加入”或复制服务器的 IP 地址，在 CS2 游戏控制台（按 `~` 键打开）输入 `connect [服务器IP]` 回车即可进入。
    

### 2\. 通过 CS 国服社区服网页进入

*   浏览器访问官方的 [CS 国服社区服大厅网页](http://csgo.wanmei.com/communityserver)。
    
*   在搜索框输入“UB”或“皮肤”进行检索。
    
*   找到目标服务器后，点击直接唤起游戏加入。
    

### 3\. 通过完美世界竞技平台进入

*   打开“完美世界竞技平台”客户端。
    
*   在顶部导航栏找到并点击 **“社区服”**。
    
*   在搜索栏搜索“UB”，选择一个服务器双击或点击加入，平台会自动启动游戏并连接到该服务器。

## 📥 如何下载与安装扩展 (ZIP压缩包)

由于浏览器的安全机制，推荐使用“开发者模式”直接加载解压文件夹的方式进行安装，百分百成功且不会被浏览器自动禁用：

### 第一步：下载并解压 ZIP 文件

1.  在本项目的上方文件列表中，找到后缀为 `.zip` 的压缩包文件（例如 `小白看皮肤_v1.0.0.zip`）。
    
2.  点击该文件名进入文件详情页。
    
3.  点击页面上的 **“下载” (Download)** 按钮，将压缩包下载到你的电脑上。
    
4.  下载完成后，**务必将该 ZIP 压缩包解压成一个普通的文件夹**。请记住这个解压后文件夹的存放位置。
    

### 第二步：安装到 Chrome 浏览器

1.  打开你的 Chrome 浏览器。
    
2.  在顶部地址栏复制并粘贴 `chrome://extensions/` 然后回车，进入“扩展程序”管理页面。
    
3.  在页面右上角，打开 **“开发者模式”** 的开关。
    
4.  开启后，页面左上角会出现几个按钮，点击最左侧的 **“加载已解压的扩展程序”**。
    
5.  在弹出的选择窗口中，找到并选中你刚才解压出来的那个文件夹（**注意**：是选中整个文件夹，不要点进去选里面的单个文件），然后点击“选择文件夹”。
    
6.  安装成功！你的扩展列表中会立刻出现「小白看皮肤」的卡片。
    

> **💡 温馨提示**：安装成功后，请不要删除你解压出来的那个文件夹，否则插件会失效。建议刷新一下 BUFF 或悠悠有品的网页，让脚本立刻生效！

## 💻 进服后如何使用指令？（控制台操作指南）

当你在网页上点击了“✅ 指令已复制”，并且成功进入了 UB 等支持检视的 CS2 社区服后，只需要简单的几步就能立刻把皮肤“穿”在身上：

### 1\. 开启开发者控制台（如果尚未开启）

*   打开 CS2 游戏，进入 **“设置”** (左上角齿轮图标) -> **“游戏”** 设置页。
    
*   找到 **“启用开发者控制台 (~)”** 选项，将其设置为 **“是”**。
    

### 2\. 呼出控制台并发送指令

*   在社区服的游戏对局中，按下键盘左上角的 `~` 键（就在 Esc 键的正下方），此时屏幕上会弹出一个灰色的控制台窗口。
    
*   在控制台最下方的输入框里，按下 `Ctrl + V`，粘贴你刚才从网页上复制的指令（比如 `sm_skin ak47 ...`）。
    
*   按下 `Enter` (回车键) 发送指令。
    

### 3\. 尽情检视欣赏！

*   再次按下 `~` 键关闭控制台。
    
*   此时你手中的武器、手套或游戏人物，就已经瞬间变成你刚才相中的饰品了！
    
*   长按键盘上的 `F` 键，尽情地在游戏光影下检视它的磨损和印花刮蹭效果吧！

## 💡 常见问题与注意事项 (FAQ)

**Q: 为什么输入指令后，手套偶尔加载不出来、显示错误，或者看起来很模糊？**

**A:** 这是 CS2 社区服插件和游戏引擎换肤机制的常见现象。当你第一次切换到某款特定的手套时，服务器和本地客户端需要短暂的时间来加载并渲染高清材质贴图。

> **👉 解决办法：** 遇到这种情况不用担心，只需在原地等待几秒钟，或者再次打开控制台，按键盘上的 `↑` 键（自动调出你刚刚输入的上一条指令），重新回车发送一次即可。手套马上就会恢复完美的无缝切换和高清状态！
> 实在不行就按键盘上的字母“M”换个边，包解决问题的！
